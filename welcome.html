HTML


HTML (HyperText Markup Language) is the standard markup language used to create and design webpages. It forms the basic structure of web content, enabling browsers to display text, images, and other elements in a coherent and organized manner.it is used for creating and structuring content on the web. It provides the foundational elements that define the structure of a webpage, which web browsers then interpret and display to users.

CSS


CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation and layout of web pages. It allows developers to separate content from design, making it easier to maintain and update the look and feel of a website. CSS works by associating style rules with HTML elements, which can include specifications for colors, fonts, spacing, positioning, and responsive behaviors. Each rule consists of a selector and a declaration block.

BOOTSTRAP


Bootstrap is a popular open-source front-end framework for developing responsive and mobile-first web applications. It contains CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components. Developed by Twitter, Bootstrap simplifies the process of creating professional-looking web pages and applications.
